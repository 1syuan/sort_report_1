插入排序法 (Insertion Sort)

直觀的排序方式，就像理牌一樣

回到首頁

📖 操作原理

插入排序法的工作原理是將序列分為「已排序」和「未排序」兩部分。每一輪從未排序部分取出一個元素，將其插入到已排序部分的適當位置。

  • 步驟 1：從第二個元素開始（假設第一個元素已排序）。
  • 步驟 2：取出該元素，與左側已排序的元素逐一比較。
  • 步驟 3：若左側元素較大，則將其右移。
  • 步驟 4：找到正確位置後，將元素插入。

📊 演算法複雜度

時間複雜度 (平均/最差)
O(n²)
當陣列完全反序時效率最低
時間複雜度 (最佳)
O(n)
當陣列已經排序好時
空間複雜度
O(1)
原地排序，僅需一個額外空間暫存當前元素

💻 插入排序法程式碼 (JavaScript)

function insertionSort(arr) { let n = arr.length; for (let i = 1; i < n; i++) { let key = arr[i]; let j = i - 1; // 將大於 key 的元素向右移動 while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j = j - 1; } // 將 key 插入正確位置 arr[j + 1] = key; } return arr; }
