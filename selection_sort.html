選擇排序法 (Selection Sort)

簡單直觀的搜尋式排序

回到首頁

📖 操作原理

選擇排序法的原理是：每一輪從「未排序部分」中尋找最小的元素，並將它與未排序部分的「第一個元素」交換位置。

  • 步驟 1：在未排序序列中找到最小元素。
  • 步驟 2：將該元素與未排序序列的起始位置交換。
  • 重複：移動到下一個位置，重複上述過程，直到整個陣列排序完成。

📊 演算法複雜度

時間複雜度 (最佳/最差)
O(n²)
不論是否已排序，皆需掃描剩餘元素
空間複雜度
O(1)
原地排序，僅需極少額外空間

💻 選擇排序法程式碼 (JavaScript)

function selectionSort(arr) { let n = arr.length; for (let i = 0; i < n - 1; i++) { let minIdx = i; // 尋找從 i 到 n-1 中最小的元素 for (let j = i + 1; j < n; j++) { if (arr[j] < arr[minIdx]) { minIdx = j; } } // 將找到的最小元素與位置 i 的元素交換 if (minIdx !== i) { let temp = arr[i]; arr[i] = arr[minIdx]; arr[minIdx] = temp; } } return arr; }
